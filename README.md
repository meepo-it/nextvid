# TanStack Starter

Ship Faster, Cost Less

TanStack Starter is the complete TanStack Start boilerplate for building profitable SaaS, packed with AI, auth, payments, storage, email, newsletter, blog, dashboard, SEO and more, fully deployed on Cloudflare Workers.

## Author

This project is created by [Fox](https://x.com/indie_maker_fox), the founder of [TanStack Starter](https://tanstackstarter.dev), [MkSaaS](https://mksaas.com) and [Mkdirs](https://mkdirs.com).

## Documentation

The documentation is available on the [website](https://docs.tanstackstarter.dev). It includes guides, tutorials, and detailed explanations of the code. I designed it to be as beginner-friendly as possible, so you can start making money from day one.

If you found anything that could be improved, please let me know.
